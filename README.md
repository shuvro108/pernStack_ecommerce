# TerraCotta - A simple eCommerce website

TerraCotta is an open-source **Next.js eCommerce frontend** project.  
It provides a modern, fast and customizable shopping UI.  
This repo is **frontend only** â€“ contributors can improve the design, add new pages, animations and more.

---

## Features

- Built with **Next.js + Tailwind CSS**
- Responsive design
- Reusable components
- Customizable layouts and colors
- Open for contributions (UI/UX, animations, themes, layouts etc.)

---

## Getting Started

1. Clone the repo

   ```bash
   
   cd TerraCotta
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Run locally

   ```bash
   npm run dev
   ```
